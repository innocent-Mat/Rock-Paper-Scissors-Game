# Rock, Paper, Scissors Game

## Overview

This project is a browser-based implementation of the classic Rock, Paper, Scissors game. Players can select Rock, Paper, or Scissors, and the computer randomly selects its choice. The game automatically determines the winner of each round, updates scores, and declares the first participant to reach three points as the overall winner.

## Features

* Play against a computer opponent.
* Random computer choices using JavaScript.
* Automatic round winner detection.
* Real-time score tracking.
* Winner announcement when a score of 3 is reached.
* Reset game functionality.
* Dynamic user interface updates using DOM manipulation.

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## How to Play

1. Click one of the three options:

   * Rock
   * Paper
   * Scissors
2. The computer will randomly make its selection.
3. The winner of the round is displayed.
4. Scores are updated automatically.
5. The first player to reach 3 points wins the game.
6. Click the **Play Again?** button to reset and start a new game.

## Game Rules

* Rock beats Scissors.
* Scissors beats Paper.
* Paper beats Rock.
* Matching choices result in a tie.
